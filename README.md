# Lumen Notes website

Static landing page for GitHub Pages.

## How to use
1. Upload all files in this folder to a GitHub repository.
2. Go to Settings -> Pages.
3. Choose Deploy from branch -> main -> root.
4. Save.

## Before launch
- Replace Instagram link in `index.html`.
- Replace phone/WhatsApp link in `index.html`.
- Add final domain and email when available.
- If you own licensed font files for Acumin Pro Wide and Latin Modern Mono Light, add them locally and update `style.css`. Do not upload font files unless your license allows web use.
